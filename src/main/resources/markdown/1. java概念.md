### Java 基础
1. Java概述：Java是一种可以撰写跨平台应用程序的面向对象的程序设计语言。Java技术具有卓越的通用性，高效性，平台移植性和安全性，广泛应用于PC，数据处理，游戏控制，移动电话和互联网，同事拥有全球最大开发者群体。
2. Java历史：Java最初成为Oak，1991年为消费类电子产品嵌入式芯片设计，1995年更名Java,设计用于Internet应用。
3. 组成：Java由四方面组成：
	1. Java语法。
	2. Java文件格式。
	3. Java虚拟机（JVM）。
	4. Java应用程序接口（Java API）。
4. 体系：三个体系：
	1. Java SE（J2SE，Java2 Platform Standard Edition，标准版）
	2. JavaEE（J2EE，Java 2 Platform, Enterprise Edition，企业版）
	3. Java ME（J2ME，Java 2 Platform Micro Edition，微型版）
5. 优势：Java 编程语言是个简单、面向对象、分布式、解释性、健壮、安全与系统无关、可移植、高性能、多线程和静态的语言。
