### Java 基础
1. Java概述：Java是一种可以撰写跨平台应用程序的面向对象的程序设计语言。Java技术具有卓越的通用性，高效性，平台移植性和安全性，广泛应用于PC，数据处理，游戏控制，移动电话和互联网，同事拥有全球最大开发者群体。
2. Java历史：Java最初成为Oak，1991年为消费类电子产品嵌入式芯片设计，1995年更名Java,设计用于Internet应用。
3. 组成：Java由四方面组成：
	1. Java语法。
	2. Java文件格式。
	3. Java虚拟机（JVM）。
	4. Java应用程序接口（Java API）。
4. 体系：三个体系：
	1. Java SE（J2SE，Java2 Platform Standard Edition，标准版）
	2. JavaEE（J2EE，Java 2 Platform, Enterprise Edition，企业版）
	3. Java ME（J2ME，Java 2 Platform Micro Edition，微型版）
5. 优势：Java 编程语言是个简单、面向对象、分布式、解释性、健壮、安全与系统无关、可移植、高性能、多线程和静态的语言。
	
6. Java 各个版本特性
	1.Java 5新特性：
		1. 泛型和变长参数。
		2. for-each循环
		3. 自动装箱机制。
		4. 字符串连接。
		5. 枚举。
		6. 注解。
	2. Java 6新特性：
		1. Instrumentation 类定义动态改变和操作。
		2. HTTP增强。
		3. JMX与系统管理。
		4. 编译器API。
		5. JavaDB 和JDBC 4.0
		6. 对脚本语言的支持。
		7. XML API 与 Web服务。
	3. Java 7新特性：
		1. 支持动态语言类型。
		2. G1 垃圾回收器。
		3. 语言改进： 支持String的switch，try-with-resources,更简洁泛型，多重catch。
		4. 核心类库改进：classLoader API , java.util.concurrent,国际化。
		5. I/O 与网络
			1. 新的文件系统API支持大连文件属性访问，系统监控服务等。
			2. Socket和文件的异步I/O。
			3. Socket channel 功能完善，支持bingding，多播等。
			4. 支持 zip/jar 的 FileSystemProvider 实现
			5. SDP(Socket Direct Protocol)
			6. 使用 Windows Vista 上的 IPv6 栈
